Telegram bot

В проекті використовувались навички створення ботів для месенджера Telegram. Ідея бота, здійснення запиту шляхом поданя ключового слова "вйо" та відповідь на нього, в нашому випадку, анекдотом. Для реалізації цього проекту використовувались: Phyton3.7, бібліотека pyTelegramBotAPI, функція randint із бібліотеки Random. Анекдоти отрмували, шляхом парсингу з інтернету. 

        
    # -*- coding: utf-8 -*-
    import telebot
    from random import randint

    bot = telebot.TeleBot("your-bot-token")

    # -*- coding: utf-8 -*-
import telebot
import requests

bot = telebot.TeleBot("your-bot-token")

def random_joke():
	url = "http://api.icndb.com/jokes/random"
	joke = requests.get(url).json()
	return joke['value']['joke']

@bot.message_handler(content_types=["text"])
def any_msg(message):
	if message.text.lower() == "вйо":
		bot.send_message(message.chat.id, random_joke())
	else:
		bot.send_message(message.chat.id, 'message')

if __name__ == '__main__':
	bot.polling(none_stop=True)
